About Legacy Project.

Proudly run by fans.

Wild Ones Legacy is the unique real Wild Ones Experience.

We use the 2012 Wild Ones original client. In other words, we are using the same client that Playdom used in 2012.

We are proudly run by fans.

We were players of the old Playdom game. Now, our experienced developer team focus on bringing all the original features of the old game.
